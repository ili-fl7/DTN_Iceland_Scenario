This folder, contains all the configuration files for Iceland context with minimum density of cars.
A day is divided into four day time scenarios: morining(0), noon(1), evening(2), and night(3).
Configuration for morning is on April 11th 2010 while the rest cover 10th.
There is only one mobility model for morning and night.
There are two mobility models for noon and evening based on the minimum and maximum group 'B'.
Configuration files must be passed as arguments when running the simulation, e.g.:
one.bat [configuration_file_name.txt] for GUI mode or
one.bat -b [number of runs] [configuration_file_name.txt] for batch mode.

** NOTE **
Delete the default configuration file from the root directory before using these custom settings.
