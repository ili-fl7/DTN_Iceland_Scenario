This folder, contains all the configuration files for Iceland context.
A day is divided into four day time scenarios: morining(0), noon(1), evening(2), and night(3).
There is only on configuration file for morning and night.
There are two configuration files for noon and evening based on the mobility model.
Configuration files must be passed as arguments when running the simulation, e.g.:
one.bat [configuration_file_name.txt] for GUI mode or
one.bat -b [number of runs] [configuration_file_name.txt] for batch mode.
